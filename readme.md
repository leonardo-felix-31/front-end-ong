# Plataforma Web de ONG Fictícia

## 🏛 Contexto
O terceiro setor brasileiro é uma força social e econômica significativa, movimentando bilhões e empregando milhões de pessoas. Muitas ONGs ainda não possuem presença digital adequada, o que limita seu alcance e captação de recursos.  
Este projeto busca desenvolver uma plataforma web profissional para gerenciar atividades, divulgar projetos, engajar voluntários e captar recursos.

## 🎯 Objetivos
- Aplicar fundamentos de HTML5 na criação de estruturas semânticas complexas;
- Implementar layouts responsivos com CSS3 avançado;
- Desenvolver funcionalidades dinâmicas com JavaScript;
- Criar formulários interativos e validados;
- Produzir documentação técnica de qualidade.

## 🖥 Páginas
1. **index.html** – Página inicial, apresenta a ONG, missão, visão, valores e informações de contato;
2. **projetos.html** – Mostra os projetos sociais da ONG e oportunidades de voluntariado;
3. **cadastro.html** – Formulário de cadastro de voluntários, com validação nativa HTML5.

- **HTML5** (estrutura semântica completa)

## 📝 Formulário de Cadastro
- Campos: Nome Completo, E-mail, CPF, Telefone, Data de Nascimento, Endereço, CEP, Cidade, Estado  
- Validação nativa com HTML5  
- Máscaras aplicadas em CPF, telefone e CEP
